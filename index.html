How to Call the API from Postman

  1. Open Postman and create a new request
  2. Set the request type to POST or GET
  3. Enter your API URL, for example: https://ap-itest-eight.vercel.app//api/messages
  4. Add Basic Auth credentials: username admin, password password123
  5. For POST requests, set the body to {"text": "Your message"}
  6. Click Send to execute the request

Response example:

{
  "text": "Your message",
  "timestamp": "2025-04-05T15:06:04.686Z"
}
